# yProHub Android (Capacitor)

This is a standalone Android codebase scaffolded with Capacitor.
It is intentionally isolated from the existing web UI (`prohub-ui`) and API (`ProHubAPI`).

## Folder purpose

- `www/` - Web assets loaded by the Android shell.
- `android/` - Native Android project generated by Capacitor.
- `capacitor.config.json` - Capacitor app config.

## Commands

Run from this folder:

```bash
npm install
npm run web:sync
npm run cap:sync
npm run cap:open
```

Optional:

```bash
npm run cap:copy
npm run android
```

## Integrating existing Angular web build

When you are ready to use your real web app in Android:

1. Run `npm run web:sync` from this folder.
2. Open Android Studio with `npm run cap:open` and build/sign APK or AAB.

`web:sync` performs all required steps:
- Builds Angular from `../prohub-ui`.
- Copies `../prohub-ui/dist/prohub-ui/browser/*` into `www/`.
- Syncs Capacitor Android assets.

This keeps mobile packaging separate from your web and backend codebases.
